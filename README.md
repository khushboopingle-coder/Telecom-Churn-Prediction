# 📊 Telecom Customer Churn Prediction

## 🔍 Problem Statement
Predict which telecom customers are likely to churn using 
machine learning and visualize insights using Power BI.

## 📁 Dataset
- Source: Kaggle Telco Customer Churn Dataset
- Size: 7,043 customers | 21 features

## 🛠️ Tools & Technologies
- Python (Pandas, Scikit-learn, Matplotlib, Seaborn)
- Random Forest Classifier
- Power BI (Interactive Dashboard)
- Google Colab

## 🤖 Models Used
| Model | Accuracy |
|-------|----------|
| Logistic Regression | 78.77% |
| Random Forest | 79.56% |

## 📊 Key Insights
- Overall churn rate: **26.54%**
- Month-to-month contracts have **highest churn**
- Fiber optic users churn **more than DSL**
- Electronic check payment has **highest churn rate**
- Gender has **almost no impact** on churn (50/50)
